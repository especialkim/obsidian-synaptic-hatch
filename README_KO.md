# Synaptic Hatch

**당신의 생각을 담는 작은 포털**

Obsidian의 경계를 넘어, 가장 중요한 노트를 항상 당신 곁에 둘 수 있습니다. 웹서핑을 하면서, 코딩을 하면서, 혹은 다른 문서 작업을 하면서 갑자기 떠오른 아이디어나 중요한 정보를 놓치지 마세요. **Synaptic Hatch**는 어떤 노트 창이든 화면 최상단에 고정(Always-on-Top)할 수 있게 해주는 플러그인입니다.

> ⚠️ **데스크톱 전용**: 이 플러그인은 운영체제의 창 제어 기능을 사용하므로 Windows, macOS, Linux 환경에서만 작동합니다.

---

## ✨ 이런 분들께 추천합니다

### 📝 라이트 유저
- 화상회의 중 회의록을 최상단에 띄워두고 싶으신 분
- 참고 자료를 다른 앱 위에 항상 보이게 하고 싶으신 분
- 듀얼 모니터에서 레퍼런스 노트를 고정해두고 싶으신 분

### 🚀 파워 유저
- Raycast, Alfred 같은 런처 도구로 특정 노트를 즉시 팝업하고 싶으신 분
- 하루를 시작할 때 단축키 하나로 오늘의 일간 노트를 띄우고 싶으신 분
- 프로젝트별로 빠른 메모 창을 만들어 퀵노트 앱처럼 사용하고 싶으신 분

## 🎯 주요 기능

### 기본 기능
- **창 고정 토글**: 클릭 한 번, 단축키 하나로 현재 창을 화면 최상단에 고정하거나 해제
- **빠른 팝아웃**: 보고 있는 노트를 새로운 항상-위 팝업 창으로 즉시 띄우기
- **고정 상태 표시기**: 어떤 창이 고정되어 있는지 직관적으로 알 수 있는 핀(📌) 아이콘
- **배경 모드**: 메인 창은 자동으로 뒤로 보내고 팝아웃만 전면 유지

### 고급 기능 (파워 유저용)
- **커스텀 팝아웃 명령어**: 원하는 파일, 폴더, 저널 노트를 한 번에 여는 나만의 명령어 생성
- **Obsidian URI 지원**: `obsidian://` 링크로 외부 런처나 자동화 도구와 연동
- **Periodic Notes 연동**: 일간/주간/월간/분기/연간 저널을 즉시 팝아웃으로 열기

## 🚀 빠른 시작

### 1단계: 기본 사용법 익히기

#### 창 고정하기
가장 기본적인 사용법입니다.

1. `Ctrl/Cmd + P`를 눌러 커맨드 팔레트를 엽니다
2. `창 고정 토글`을 검색해 실행합니다
3. 현재 창이 화면 최상단에 고정됩니다 (다시 실행하면 해제)

**또는** 창 우측 상단의 📌 아이콘을 클릭하세요 (설정에서 표시기를 활성화한 경우)

#### 노트를 팝아웃으로 띄우기
지금 보고 있는 노트를 빠르게 팝업 창으로 만들 수 있습니다.

1. `Ctrl/Cmd + P`로 커맨드 팔레트를 엽니다
2. `현재 파일을 새 고정 창에서 열기`를 실행합니다
3. 현재 노트가 즉시 최상단 고정된 팝업 창으로 나타납니다

> **💡 팁**: 자주 쓰는 커맨드는 단축키를 지정하세요! `설정 → 단축키`에서 원하는 키 조합을 할당할 수 있습니다.

---

### 2단계: 나만의 커스텀 팝아웃 만들기 (선택사항)

이 기능은 Synaptic Hatch의 **진짜 힘**입니다. 여러분의 워크플로우에 딱 맞는 명령어를 직접 만들 수 있습니다.

#### 설정 열기
1. `설정 → 커뮤니티 플러그인 → Synaptic Hatch`로 이동
2. **커스텀 팝아웃 명령어** 섹션의 토글을 켭니다
3. `+ 명령어 추가` 버튼을 클릭합니다

#### 명령어 타입 선택하기

| 타입 | 언제 사용하나요? | 설정 예시 |
| :--- | :--- | :--- |
| **📄 파일** | 특정 노트를 항상 같은 방식으로 열고 싶을 때<br/>예: 작업 체크리스트, 프로젝트 대시보드 | `Tasks/Inbox.md` 입력 |
| **📁 폴더** | 매일/매시간 새로운 노트를 만들고 싶을 때<br/>예: 회의록, 업무 일지 | 폴더: `Meeting Notes`<br/>파일명 규칙: `{{date}}_회의`<br/>템플릿: `Templates/회의록.md` (선택) |
| **📅 저널** | Periodic Notes로 일간/주간 노트를 관리할 때<br/>예: 데일리 노트, 주간 리뷰 | 주기 선택: 일간/주간/월간/분기/연간 |
| **📋 빈 창** | 빠른 메모용 빈 창이 필요할 때<br/>예: 전화 메모, 임시 스크래치 | 설정 불필요 |

#### 명령어 활성화하기
1. 필요한 정보를 모두 입력했다면
2. 오른쪽의 **토글 스위치**를 켭니다
3. 커맨드 팔레트에서 `Custom Popout: ...` 형태로 나타납니다!

---

### 3단계: 외부 도구와 연동하기 (파워 유저)

이제 여러분만의 퀵노트 앱을 만들 차례입니다!

#### URI 복사하기
1. 활성화된 커스텀 명령어 옆의 **🔗 복사 버튼**을 클릭
2. `obsidian://custom-popout?vault=...` 형식의 링크가 복사됩니다

#### 활용 예시

**Raycast (Mac)**
```
1. Raycast 실행 → "Quicklinks" 검색
2. 새 Quicklink 생성
3. 이름: "업무 일지"
4. 링크: (복사한 URI 붙여넣기)
5. 이제 Raycast에서 "업무 일지"를 검색하면 즉시 팝업!
```

**Alfred (Mac)**
```
1. Workflows → 새 Blank Workflow
2. Hotkey Trigger 추가 (예: Cmd+Shift+Space)
3. "Open URL" 액션 연결 → (복사한 URI 붙여넣기)
4. 이제 단축키로 언제든지 노트 창 호출!
```

**Stream Deck / 전역 단축키 도구**
- 복사한 URI를 버튼이나 단축키에 할당하면 됩니다!

## ⚙️ 설정 가이드

### 표시기 (Indicator) 커스터마이징

#### 메인 창 표시기 / 팝아웃 창 표시기
각각 독립적으로 켜고 끌 수 있습니다.

- **위쪽 오프셋**: 표시기를 창 위에서 얼마나 떨어뜨릴지 (픽셀)
- **오른쪽 오프셋**: 표시기를 창 오른쪽에서 얼마나 떨어뜨릴지 (픽셀)
- **표시기 크기**: 배경 원의 크기 (20-60px)
- **아이콘 크기**: 핀 아이콘의 크기 (10-50px)

> **💡 팁**: 표시기가 다른 버튼과 겹친다면 오프셋 값을 조정해보세요!

### 날짜 형식

커스텀 팝아웃에서 `{{date}}`를 사용할 때 적용되는 형식입니다.

- 기본값: `YYYY-MM-DD` → `2024-03-15`
- 다른 예시:
  - `YYYY년 MM월 DD일` → `2024년 03월 15일`
  - `YYYYMMDD_HHmmss` → `20240315_143022`

## 💡 실전 활용 팁

### 듀얼 모니터 사용자
두 번째 모니터에 팝아웃 창을 띄워 참고 자료를 항상 최상단에 두고, 메인 모니터에서 작업하세요. 시선 이동이 최소화됩니다.

### 화상회의 / 온라인 강의
회의 앱 화면을 보면서도 Obsidian 팝아웃으로 필기하면 화면 전환 없이 메모할 수 있습니다.

### 글쓰기 / 연구
여러 레퍼런스 노트를 각각 팝아웃으로 띄워두고, 메인 창에서 글을 작성하세요. 여러 창 사이를 오가는 수고를 덜 수 있습니다.

### 아침 루틴 자동화
1. 커스텀 팝아웃에서 "일간 저널" 명령 생성
2. URI를 복사
3. 아침에 컴퓨터를 켜면 자동으로 실행되도록 설정 (예: Mac의 Automator, Windows의 작업 스케줄러)
4. 하루를 노트와 함께 시작하세요!

### 프로젝트별 퀵메모
각 프로젝트마다 "빠른 메모" 폴더 명령을 만들고, 단축키를 할당하세요. 작업 중 떠오른 생각을 즉시 기록할 수 있습니다.

## 🔧 문제 해결

| 증상 | 해결 방법 |
| --- | --- |
| 창 고정이 작동하지 않아요 | 다른 프로그램이 Always-on-Top을 가로채는지 확인하세요. Obsidian을 재시작해보세요. |
| 팝아웃 창이 바로 닫혀요 | Obsidian 0.15.0 이상이 필요합니다. 버전을 확인하세요. |
| 커스텀 명령이 안 보여요 | 1. 설정에서 명령이 **활성화**되어 있나요?<br/>2. 파일 경로나 폴더 경로가 올바른가요?<br/>3. 파일명 규칙에 금지된 문자(`/`, `\`, `:` 등)가 없나요? |
| URI가 동작하지 않아요 | Vault 이름에 한글이나 공백이 있으면 자동으로 인코딩됩니다. URI를 직접 수정하지 마세요. |
| 표시기가 다른 버튼과 겹쳐요 | 설정에서 "오프셋" 값을 조정하여 위치를 옮겨보세요. |

## 🎓 시작하기 전 꼭 알아두세요

### 파일 경로는 어떻게 입력하나요?
- Obsidian Vault의 루트 폴더 기준 상대 경로를 입력하세요
- 예: `Projects/Work/Dashboard.md`
- 폴더를 선택할 때는 입력하면 자동완성이 됩니다!

### `{{date}}` 치환은 언제 사용하나요?
- 폴더 타입 커스텀 팝아웃에서 사용합니다
- 파일명 규칙에 `{{date}}`를 넣으면 현재 날짜로 자동 치환됩니다
- 예: `{{date}}_회의록` → `2024-03-15_회의록.md`

### 저널 타입은 언제 사용할 수 있나요?
- Obsidian 코어 플러그인의 **Daily Notes**를 활성화했거나, 커뮤니티 플러그인 **Calendar**(주간 노트 지원), **Periodic Notes** 등 주기 노트를 제공하는 플러그인에 의존합니다
- 위 기능 중 하나라도 설정되어 있다면 사용 가능합니다
- 일간/주간/월간/분기/연간 중 실제로 설정된 주기만 선택할 수 있습니다

## 📦 설치 방법

### (승인 검토 중입니다) 커뮤니티 플러그인 스토어
1. Obsidian에서 `설정 → 커뮤니티 플러그인`을 엽니다
2. Safe mode를 끕니다 (처음 한 번만)
3. `Browse` 버튼을 클릭합니다
4. `Synaptic Hatch`를 검색합니다
5. `설치` 후 `활성화`를 클릭합니다

### 수동 설치

1. 아래 GitHub 저장소로 이동합니다  
   https://github.com/especialkim/obsidian-synaptic-hatch
2. **Releases** 탭에서 최신 버전의 ZIP 파일을 다운로드합니다
3. 압축을 풀고 `obsidian-synaptic-hatch` 폴더를 옵시디언 플러그인 경로 ``<vault>/.obsidian/plugins/`에 복사합니다  
4. Obsidian을 재실행한 뒤  
   `설정 → 커뮤니티 플러그인`에서 **Synaptic Hatch**를 활성화합니다
